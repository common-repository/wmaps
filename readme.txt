Enable wordpress to use google maps on your post

== Description ==

Plug-in that enable your wordpress blog to use easily Google Maps in your posts.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the 'wmaps' folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'plugins' page in WP.
4. After yo activate the plug, you will see an new option called "My Maps" in the "Settings" menu.
